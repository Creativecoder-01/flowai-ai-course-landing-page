
# FlowAI — AI Course Landing Page Template

A modern and responsive landing page for an AI productivity course, built with HTML, CSS, and JavaScript.

## Live Demo

[View Live Demo] 
https://creativecoder-01.github.io/flowai-ai-course-landing-page/

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Clean and semantic HTML structure
- Responsive design for all screen sizes
- Mobile navigation menu
- CSS variable system for easy customization
- Smooth accordion interactions (curriculum + FAQ)
- Modern dark dashboard-style UI
- Smooth hover and transition effects


## Ideal For

- AI course creators
- Online educators
- Digital product businesses
- Coaching programs
- SaaS and productivity products

## Responsive Design

The template is fully responsive and optimized for:

- Desktop devices
- Tablets
- Mobile phones

## Customization

### Colors
All colors are defined as CSS variables in `:root` inside `style.css`:

```css
--bg-primary: #0D1117;
--bg-secondary: #111827;
--accent-primary: #7C3AED;
--accent-secondary: #8B5CF6;
--color-primary: #F8FAFC;
--color-secondary: #94A3B8;
```

### Fonts
This template uses Google Fonts:
- **Plus Jakarta Sans** — headings
- **Inter** — body text

### Sections
Each section is clearly commented in `index.html` for easy editing:
- Replace text with your course content
- Swap images in the `images/` folder
- Update pricing plans to match your offering
- Edit accordion modules with your curriculum

## Installation

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Customize the content, colors, images, and text as needed.
4. Deploy to your preferred hosting platform.
